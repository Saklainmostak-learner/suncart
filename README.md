# 🌞 SunCart

A modern **summer e-commerce web application** built with Next.js, featuring authentication, product browsing, and a dynamic cart system.

---


## 📌 Features

* 🔐 User Authentication (Email & Google Login)
* 🔒 Protected Product Details Page
* 🛍️ Product Listing & Details View
* 🛒 Add to Cart System (with LocalStorage)
* 🔢 Cart Count in Navbar
* ❌ Remove & Clear Cart Items
* 👤 User Profile & Update Profile
* 📱 Fully Responsive Design
* 🎞️ Animated Hero Section & Brand Slider

---

## 🧱 Tech Stack

* ⚡ Next.js (App Router)
* 🎨 Tailwind CSS + DaisyUI
* 🔐 Better Auth
* 🗄️ SQLite (Auth Storage)
* 💾 LocalStorage (Cart System)
* 🔥 React Hot Toast

---



## ⚙️ Installation

npm install
npm run dev


---

## 🧠 How It Works

* Authentication handled by Better Auth
* Cart system stored in browser (LocalStorage)
* Protected routes redirect unauthenticated users
* Dynamic UI updates via React state
